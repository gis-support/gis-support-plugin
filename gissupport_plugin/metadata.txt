description=GIS Support Plugin is the essential bridge between QGIS and Polish spatial data infrastructure. It gives you one-click access to services and datasets published by Polish public institutions - without leaving QGIS or hunting for URLs.
about=What it does:
	* Cadastral parcel search - find any parcel in Poland by commune, precinct and parcel number using the ULDK service (GUGiK); the parcel boundary is loaded directly onto your map

	* Elevation profile - draw a line on the map and instantly generate an elevation chart using the GUGiK NMT API; individual point elevations are also available on click

	* National WMS/WFS catalogue - a curated list of country-wide map services (orthophoto, topographic maps, administrative boundaries and more) ready to add in one click - no need to remember or search for service URLs

	* WMTS with local tile caching - add tiled map services with configurable on-disk caching so downloaded tiles (e.g. orthophoto) are reused across sessions

	* Data downloads - download PRG administrative boundaries (communes, counties, voivodeships), address points and BDOT10k topographic objects for any area of Poland directly from GUGiK

	* Digital Elevation Model - bulk download NMT/NMPT raster data published by GUGiK

	* Historical map search (Mapster) - browse and preview archival maps covering your current map view

	* Usemaps Lite - a built-in cloud workspace for QGIS teams: upload layers to the cloud, share them with colleagues and keep everyone working on the same up-to-date data - no server setup required

	The plugin is free, requires no API keys and works on Windows, macOS and Linux. Maintained by GIS Support: https://gis-support.pl.

description[pl]=Wtyczka GIS Support zawiera narzędzia, które ułatwiają komunikację z usługami oferowanymi przez polskie instytucje (np. GUGiK).
about[pl]=Co potrafi wtyczka: 

    * Wyszukiwarka działek ewidencyjnych – znajdź dowolną działkę w Polsce, wpisując gminę, obręb i numer działki. Dzięki usłudze ULDK (GUGiK) granica działki zostanie od razu załadowana na Twoją mapę.

    * Profil wysokościowy GUGIK NMT - narysuj linię na mapie i błyskawicznie wygeneruj wykres wysokościowy, korzystając z API NMT (GUGiK). Możesz także sprawdzić wysokość konkretnego punktu po kliknięciu.

    * Baza krajowych usług WMS/WFS – starannie dobrana lista ogólnopolskich usług mapowych (ortofotomapa, mapy topograficzne, granice administracyjne i inne), które dodasz jednym kliknięciem. Zapomnij o kopiowaniu linków do serwerów.

    * WMTS z lokalnym buforowaniem (cache) – dodawaj kafelkowe usługi mapowe WMTS z możliwością zapisu danych na dysku. Raz pobrane kafle (np. ortofotomapa) będą dostępne szybciej w kolejnych sesjach.

    * Pobieranie danych – pobieraj granice administracyjne, punkty adresowe (PRG) oraz obiekty topograficzne BDOT10k dla dowolnego obszaru Polski bezpośrednio z zasobów GUGiK.

    * Numeryczny Model Terenu (NMT) – masowe pobieranie danych rastrowych NMT oraz NMPT publikowanych przez GUGiK.

    * Wyszukiwarka map historycznych (Mapster) – przeglądaj i wyświetlaj podgląd archiwalnych map obejmujących aktualny widok Twojej mapy.

    * Usemaps Lite – wbudowane chmurowe środowisko pracy dla zespołów QGIS. Przesyłaj warstwy do chmury, udostępniaj je współpracownikom i pracujcie na tych samych, aktualnych danych bez konieczności konfigurowania własnego serwera.

	Wtyczka jest całkowicie bezpłatna, nie wymaga kluczy API i działa na systemach Windows, macOS oraz Linux. Za jej rozwój i utrzymanie odpowiada zespół GIS Support: https://gis-support.pl.
